# SpecKit Analyze

Performs a non-destructive consistency check across all SpecKit artifacts.

Checks for:
- Alignment between spec, plan, and tasks
- Missing or contradictory requirements
- Incomplete task dependencies
- Quality issues in design artifacts

Returns a structured report with errors, warnings, and suggestions.
